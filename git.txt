Отмена на индексацию
git restore --staged shell.py

Откатитесь к предыдущей версии кода, путем создания нового коммита
git revert HEAD --no-edit

Добавление тега
git tag version_beta

Выведите список логов всех проделанных коммитов
git log --pretty=oneline

Откат к первой версии с помощью айди коммита
git reset --hard 94ea1c

Удаление тег
git tag -d version_beta
